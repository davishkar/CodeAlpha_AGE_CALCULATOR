# CodeAlpha Age Calculator

Welcome to the CodeAlpha Age Calculator project! This is a web-based age calculator created for the Web Development Internship Task 1.

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge&logo=livechat)](https://davishkar.github.io/CodeAlpha_AGE_CALCULATOR/)

## Description

This project is a web-based Age Calculator that allows users to calculate their age based on their submitted date of birth, month, and year. JavaScript's built-in date and time functions simplify the age calculation process, making it easy to determine someone's age.

## Features

- Input fields for date of birth, month, and year
- Calculate age based on input
- Responsive design for various screen sizes

## Usage

1. Clone the repository:

```bash
git clone https://github.com/davishkar/CodeAlpha_AGE_CALCULATOR.git
```

2. Open `index.html` in your web browser.
